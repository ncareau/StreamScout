StreamScout

A website and API to get stream information

![StreamScout Example](https://github.com/ncareau/StreamScout/raw/master/demo/demo.gif)

## Requirements

- Node with Yarn
- Python 3.7

## Configuration and running

### Docker

```bash
docker run -d --name streamscout \
    -p 80:80 \
    ncareau/streamscout
```
